## PSQL ON TERMINAL 🚀👩‍🚀

### `\l :`
- List all databases

### `\l+ :` 
- List all databases with more details

### `\c <database_name> :`
- Connect to a database

### `\dt :`
- List all tables

### `\d <table_name> :`
- List all columns of a table

### `\du :`
- List all users

### `\du+ :`
- List all users with more details

### `\q :`
- Quit
